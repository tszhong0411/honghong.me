## Step 1 - Download Windows 11 ISO

<Alert icon='ðŸ”—' title='Link' color='blue' radius='md'>
  [Download Windows 11
  ISO](https://www.microsoft.com/en-us/software-download/windows11)
</Alert>

1. Select Windows 11 (multi-edition ISO)
2. Click Download
3. Select your language
4. Click Confirm

![](</static/images/blog/how-to-install-win11-on-vmware/step-1(1-4).en.png>)

5. Click 64-bit Download

![](</static/images/blog/how-to-install-win11-on-vmware/step-1(5).en.png>)

## Step 2 - Create Windows 11 Virtual Machine

1. Click Create a New Virtual Machine

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(1).en.png>)

2. Select Typical
3. Next

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(2-3).en.png>)

4. Select Installer disc image file (iso)
5. Click Browse to select the ISO that your download before
6. Next

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(4-6).en.png>)

7. Select Microsoft Windows
8. Select Windows 10 x64 æˆ– Windows 10 and later x64 (If it has this option)
9. Next

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(7-9).en.png>)

10. Type the Virtual machine name, here I named it Windows 11
11. Click Browse to select the virtual machine install location, here I select E drive
12. Next

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(10-12).en.png>)

13. Type the Maximum disk size, recommended 60GB or more
14. Select Store virtual disk as a single file
15. Next

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(13-15).en.png>)

16. Finish

![](</static/images/blog/how-to-install-win11-on-vmware/step-2(16).en.png>)

## Step 3 - Customize the virtual machine

1. Click Edit virtual machine settings

![](</static/images/blog/how-to-install-win11-on-vmware/step-3(1).en.png>)

2. Select Memory
3. Select 4 GB or more
4. OK

![](</static/images/blog/how-to-install-win11-on-vmware/step-3(2-4).en.png>)

5. Click Power on this virtual machine

![](</static/images/blog/how-to-install-win11-on-vmware/step-3(5).en.png>)

## Step 4 - Install Windows 11

1. Select your preferred language
2. Next

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(1-2).en.png>)

3. Click Install now

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(3).en.png>)

7. Type `regedit`

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(7).en.png>)

8. Go to `Computer\HKEY_LOCAL_MACHINE\SYSTEM\Setup`
9. Right click Setup
10. New > Key, and name it `LabConfig`

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(8-10).en.png>)

11. New > DWORD (32-bit) Value

22. Select Drive 0
23. Next

![](</static/images/blog/how-to-install-win11-on-vmware/step-4(22-23).en.png>)
