## Bonus - Set up Traditional Chinese

<Alert icon='ðŸ”—' variant='info'>
  <AlertTitle>Link</AlertTitle>
  <AlertDescription>[Download Traditional Chinese Language Pack](/static/files/how-to-install-and-activate-vmware-16/zh_TW.zip)</AlertDescription>

1. Right-click VMware Workstation Pro 16 and click Open File Location

<Image
  src='/static/images/blog/how-to-install-and-activate-vmware-16/step-4(1).png'
  alt='Step 1'
  width={545}
  height={666}
/>

2. Go to messages

<Image
  src='/static/images/blog/how-to-install-and-activate-vmware-16/step-4(2).png'
  alt='Step 2'
  width={1143}
  height={779}
/>

3. Unzip the `zh-TW.zip` you just downloaded, then copy / move it to `messages`.

<Image
  src='/static/images/blog/how-to-install-and-activate-vmware-16/step-4(3).png'
  alt='Step 3'
  width={1143}
  height={686}
/>

4. Right-click on VMware Workstation Pro 16 and click on Properties

<Image
  src='/static/images/blog/how-to-install-and-activate-vmware-16/step-4(4).png'
  alt='Step 4'
  width={463}
  height={661}
/>

5. Add `--locale zh_TW` at the end of "Target"
6. Press OK

<Image
  src='/static/images/blog/how-to-install-and-activate-vmware-16/step-4(5-6).png'
  alt='Step 5-6'

  height={566}
